PÁRRAFOS

La etiqueta class= se puede utilizar tanto dentro de <"a">, <"spam"> y tambien dentro de <"p">

Encabezado de Parrafos y Encabezados (Todas las etiquetas son sin las comillas)

Para realizar un parrafo habra que utilizar la etiqueta <"p"> mientras que para realizar un encabezado habra que utilizar la etiqueta <"h">.

Tipos de Encabezados

Para realizar distintos tipos de encabezados se sigue utilizando la etiqueta <"h"> la unica diferencia para cambiar el tamaño seria poner un numero acompañando al "h", estos números van del 1 al 6 (De mayor a menor tamaño).

Separar lineas


Para separar mas al parrafo del encabezado que tiene en vez de la etiqueta <"p">, se utiliza la etiqueta <"br">.

Texto en negrita

Para realizar un texto en negrita no se usa <"p">, se utiliza la etiqueta <"Strong">

Texto en cursiva y parrafo

Para realizar un texto en cursiva en vez de utilizar la etiqueta <"p">, se utiliza la etiqueta <"em">.

Texto arriba o abajo

Para escribir arriba se utiliza la etiqueta <"sup"> Para escribir abajo se utiliza la etiqueta <"sub>

Span para que sirve

Se utiliza para cambiar una parte del parrafo o encabezado. Por ejemplo: Escribo aqui en azul, el ejemplo de escribir azul seria <"spam class="bg-azul-o"> Escribo aqui en azul <"spam">

Realizar cajas en color

La caja del texto es entera, para realizar esto se utiliza la clase del color dentro de la etiqueta div (ya que esta simboliza el bloque entero) Significaria que le estamos cambiando el color a todo el bloque.
